# 🚀 Visual Game Hub

Portal de jogos desenvolvido para o desafio de interatividade web.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Estética Neon)
- JavaScript Vanilla
- GDevelop (Engine dos jogos integrados)

## 🔊 Solução de Áudio
O projeto resolve a restrição de "Autoplay" dos navegadores modernos através de um gatilho de interação inicial na tela de entrada.# Visualgamehub.io
