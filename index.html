Campanha FAUX#ELEVATE: O Ataque de 25 Segundos e o Uso de VBScript Massivo

Publicado por Equipe Técnica Visual Studio & Security | 24 de Março, 2026

A operação coordenada FAUX#ELEVATE utiliza currículos falsos infectados com scripts VBScript de 224 mil linhas para infiltrar redes corporativas. O malware utiliza o WMI para focar em sistemas empresariais e exfiltrar dados via SMTP para endereços russos.

Velocidade Crítica: A infecção completa ocorre em apenas 25 segundos.

CNCERT Alerta para Vulnerabilidades no Agente Autônomo OpenClaw

Publicado por Equipe Técnica Visual Studio & Security | 24 de Março, 2026

O Centro de Resposta a Emergências de Redes da China (CNCERT) emitiu um comunicado urgente sobre o OpenClaw. Segundo o alerta, as diretrizes de segurança padrão frágeis da plataforma, somadas ao seu alto nível de privilégio no sistema, podem permitir que invasores assumam o controle total dos endpoints.

Injeção Indireta de Prompt (IDPI)

Uma das maiores ameaças identificadas é a Injeção de Prompt Cruz-Domínio (XPIA). Neste cenário, o atacante manipula o agente de IA através de conteúdos externos, como resumos de páginas web, forçando o sistema a vazar informações confidenciais ou ignorar filtros de segurança.

Riscos de Exfiltração via Apps de Mensagens

Investigações recentes revelaram que o recurso de pré-visualização de links em plataformas como Telegram e Discord pode ser explorado. O agente de IA é enganado para criar uma URL dinâmica que envia dados sensíveis do usuário diretamente para o domínio do criminoso, sem que haja necessidade de um clique manual.

Recomendação de Defesa: Evite expor a porta de gerenciamento do OpenClaw à rede pública e desative a atualização automática de novas habilidades vindas de repositórios não verificados.

Glossário Técnico

WMI (Windows Management Instrumentation)
Estrutura para administração de dados e operações em ambientes Microsoft Windows.
IDPI / XPIA
Injeção de instruções maliciosas em modelos de linguagem através de fontes de dados externas para manipular o comportamento da IA.
C2 (Command and Control)
Servidor central utilizado por cibercriminosos para comandar máquinas infectadas remotamente.
Exfiltração via SMTP
Transferência não autorizada de dados para fora de uma rede utilizando o protocolo de e-mail.
