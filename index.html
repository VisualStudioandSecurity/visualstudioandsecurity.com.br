Campanha FAUX#ELEVATE: O Ataque de 25 Segundos que Compromete Redes Corporativas

Publicado por Equipe Técnica Visual Studio & Security | 24 de Março, 2026

Uma nova e perigosa campanha de ataque, batizada de FAUX#ELEVATE, está mirando ambientes corporativos com uma precisão alarmante. O ataque utiliza currículos falsos como "bait" (isca) para iniciar uma cadeia de infecção que dura apenas 25 segundos do início ao fim.

A Mecânica do "Bait"

Os cibercriminosos distribuem e-mails de phishing contendo anexos que simulam documentos de currículo (CV). Ao abrir o arquivo, o usuário recebe uma mensagem de erro falsa indicando que o documento está corrompido. Na verdade, esta mensagem serve apenas para mascarar a execução de um script VBScript massivo com mais de 224 mil linhas de código ofuscado.

Exfiltração e Criptomineração

Uma vez ativo, o malware executa múltiplas funções simultâneas:

  • Roubo de Credenciais: Utiliza componentes como o ChromElevator para extrair senhas de navegadores baseados em Chromium e o mozilla.vbs para perfis do Firefox.
  • Exfiltração via SMTP: Os dados roubados são enviados para contas de e-mail russas (.ru) configuradas no script.
  • Mineração de Monero: O vírus instala o mservice.exe (XMRig) para sequestrar o poder de processamento da máquina e minerar criptoativos.

Camuflagem Avançada

O que torna esta campanha particularmente difícil de detectar é o abuso de infraestruturas legítimas. Os atacantes utilizam o Dropbox para armazenar seus payloads e sites WordPress comprometidos para hospedar as configurações de Comando e Controle (C2). Além disso, o malware tenta desativar agressivamente o Microsoft Defender e o UAC do Windows para limpar seus rastros forenses.

Recomendação: Empresas devem reforçar o monitoramento de execuções de VBScript e bloquear tráfego SMTP não autorizado em estações de trabalho.
