Campanha FAUX#ELEVATE: O Ataque de 25 Segundos

Por Equipe Técnica Visual Studio & Security | 24 de Março, 2026

A campanha FAUX#ELEVATE utiliza currículos falsos como isca ("bait") para comprometer ambientes corporativos em tempo recorde.

Mecânica da Infecção

  • O Bait: O usuário vê uma mensagem de erro de arquivo corrompido, disfarçando um script VBScript de 224 mil linhas.
  • Velocidade: A infecção completa, do início até a exfiltração de dados, leva aproximadamente 25 segundos.
  • Exfiltração: Credenciais são enviadas via SMTP para domínios russos como olga.aitsaid@mail.ru.
  • Persistência: O malware desativa o Microsoft Defender e altera o registro do Windows para desativar o UAC.
Proteção Ativa: O malware utiliza o driver WinRing0x
